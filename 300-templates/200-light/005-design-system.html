Layout Patterns Available

Choose a numbered template (010–250) as your starting point. Each has a distinct layout:

010: Classic Title
Centered title with eyebrow and divider. Best for title/cover slides.
020: Title + Body Divider
Title with horizontal rule, then content below. Separates context from body.
030: Split Diagram + Text
Left half for diagrams/images, right half for text. For technical/architecture slides.
040: Image Focus
Full-area image with minimal caption. Let images breathe.
050: Three Columns
Three equal-width cards. Ideal for "pillars" or feature highlights.
060–250: See templates
Quote, stats, timeline, code, comparison, consulting-style, and more. Browse the files.

How to Create New Slides

1

Choose a layout pattern. Pick a template file (e.g., 030-split-diagram-text.html) that matches your content structure.

2

Copy and rename. Duplicate the template, rename to your slide number (e.g., slide15.html), and place it in your slide deck folder.

3

Update content. Replace placeholder text, adjust headings, and fill in your data. Keep the CSS unchanged unless you need customisation.

4

Colour override (optional). If using a different accent colour, update the hex codes in the <style> section. All accents in this theme use #2563eb as the base.

5

Add speaker notes. Include <aside class="notes"> element at the bottom. Press N in the viewer to reveal notes.

Quick Reference: CSS Patterns

Reusable CSS snippets used across all light templates:

Flexible sizing: clamp(1rem, 2.5vw, 2rem) — scales between breakpoints
Responsive grid: grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))
Accent text: Wrap key words in <span class="accent">, style with colour #2563eb
Readability: line-height 1.6–1.8 for body, max-width ~760px for text blocks
Hierarchy: Use letter-spacing 0.08–0.2em for uppercase labels
