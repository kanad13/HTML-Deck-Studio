# Documentation Guide

Read these docs in order if you are new to the repo.

1. `100-user-guide.md` - open decks, present slides, use notes, and understand the viewer.
2. `110-pdf-export.md` - set up and run PDF export.
3. `120-security-model.md` - understand what stays local, what is sandboxed, and where the boundaries are.
4. `130-development-and-tests.md` - maintain the repo, run checks, and update CI/tests with features.

Use `300-templates/` when you are creating or maintaining generated decks with an AI agent.
